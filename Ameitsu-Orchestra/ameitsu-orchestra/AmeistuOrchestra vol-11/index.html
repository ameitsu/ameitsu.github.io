# 废旧电池的回收：一个综述

# 1. 电池的历史、发展和废旧电池问题

自从 1799 年伏特制作出第一种电池 —— 伏特电堆 —— 以来，电池已经服务人类了近两个世纪。
为使电池更加可靠和高效，无数研究者前仆后继。1836 年，英国的丹尼尔对 “伏特电堆” 进行了改良：他使用稀 H2SO4 作电解液，解决了电池极化问题，制造出第一个不极化，能保持平衡电流的锌 — 铜电池。在 1860 年，法国的雷克兰士制作出了所谓的 “湿电池”，其正极是 MnO2 和碳的混合物，负极棒和正极杯都被浸在作为电解液的 NH4Cl 溶液中。1887 年，英国人赫勒森发明了最早的干电池。在电池领域的关键性进展中，以锂离子电池为代表的二次电池具有重要地位。这使得电池的体积有效减小，扩大了其使用的领域与产量。

随着电池产量的增高，电池的废弃量也在增加：其中既有锌锰干电池等一次电池无法重复使用的因素，也有二次电池寿命耗尽的问题存在。废旧电池也逐渐成为环境保护与可持续发展中的一个重要议题。

废旧电池既是危害，更是资源。例如作为锂离子电池的主要正极材料，LiCoO2 得到了广泛的应用，但作为重金属盐，它可引起胃肠功能紊乱，耳聋、心肌缺血等症状。同时 LiCoO2 燃烧或受热分解时产生有毒的锂钴氧化物，会引起重金属污染，导致环境的 pH 值升高。与此同时，当下 LiCoO2 的价格接近十七万元每顿，而国内的 LiCoO2 供应存在缺口，需要进口矿石。若将废弃电池中的 LiCoO2 进行回收，LiCoO2 的价格与供应问题将有望得到解决。因此，处理废旧电池迫在眉睫。

# 2. 化学法处理废旧电池

2.1 锂离子电池
作为二次电池的重要种类，锂离子电池的回收一直是电池领域的热点。锂离子电池的核心目标是获取钴。分离出各种金属元素的常用工艺有酸碱溶解浸出法、化学沉淀和盐析法、离子交换法、萃取法以及电沉积法等，本文将着重介绍酸碱溶解浸出法。

# 2.1.1 酸碱浸出法研究现状

在酸碱浸出法 [1] 中，Li 等先用超声清洗分离集流体与活性物质，并使用高浓度盐酸浸出，沉淀回收钴；Liu 等采用有机溶剂浸泡极片，使集流体与活性材料分离，通过高温固相法重新合成 LiCoO2；Lee 等使用两级热处理两级过筛后高温煅烧的方法预选活性物质，分别采用 HNO3 和 HNO3 – H2O2 体系对筛后活性物质浸出；Wang 等采用盐酸浸出，KMnO4 氧化沉锰，丁二酮肟萃取镍，NaOH 沉淀回收钴。但这些研究实际应用效果欠佳，一般不具备经济可行性。造成这一局面的原因一方面是废旧锂离子电池拆分过程效率低，集流体与活性物质分离过程能耗大，成本较高，环境污染严重；另一方面是 LiMn2O4 等材料的加入使分离其他金属元素的工艺也需要兼顾。

# 2.1.2 物理擦洗−稀酸搅拌浸出法 [1]

该工艺由陈亮等人提出，可回收废旧锂离子电池中 95% 的钴、镍和锰以及铜和铁等常见金属材料。该工艺由活性物质与集流体分离、还原酸浸、浸出液的净化和三元锂离子电池正极材料前躯体的制备四步组成。
对于集流体分离，论文提出了将破碎后的电芯碎片加入热水中机械搅拌，过滤
烘干后进行第一次振动筛分并取筛上碎料的操作 。分离后即可将碎料与稀 H2SO4 中搅拌，并且最适宜的稀 H2SO4 浓度为 0.5mol/L。此时会有少量的铝和铜浸出，可直接回收，夹带出的材料可继续参与后续的回收操作。此步骤中发生的反应为

2Al+3H2SO¬4 → Al2(SO4)3+3H2↑
4LiCoO2+6H2SO4 → 2Li2SO4+4CoSO4+6H2O+O2↑

通过集流体分离获取活性物质后，废料即可进行还原酸浸。酸浸所需的试剂是 H2SO4 和 H2O2 溶液。研究者发现影响酸浸的因素有反应时间、反应温度、H2O2 溶液浓度和液固比有关，且均与镍、钴和锰的浸出率成正比。最佳浸出条件如下：2.5 mol/L H2SO4、2.0 mL H2O2/g 粉料、120min、85℃、液固比 10:1。此时，钴、镍和锰的浸出率分别为 97%、98% 和 96%，终点 pH 值为 0.5~0.8。该步骤发生的反应为

2LiCoO2+3H2SO4+H2O2 → 2CoSO4+O2↑+ Li2SO4+4H2O

完成酸浸后，进入浸出液净化步骤。其他研究表明，采用黄铁酸钠 [NaFe33 (SO4) 2 (OH) 6] 法除铁时，钴的回收率较高，铁的除去率为 99.9%，而钴、镍和锰的损失率均小于 1% ，因此是可行的方法。论文中又指出用 N902 萃取剂萃取除铜方法，可除去料液中 99.5% 的铜。为了同时除去铝，应控制溶液 pH 为 4.3 左右。该步骤发生的反应为

Fe2(SO4)3 + 12H2O + Na2SO4 → Na2Fe6(SO4)4(OH) 12↓+ 6H2SO4

净化后，即可根据料液中钴、镍和锰的物质的量之比，配入适当的 NiSO4、MnSO4 或 CoSO4，随后往溶液中添加 NH4HSO4，采用共沉淀法制备镍钴锰碳酸盐前躯体。

# 2.1.3 全湿浸出法 [2]

全湿法是另一种酸碱浸出法的改良，由吴芳等人提出。该工艺的钴的直收率达到 99％ 以上，并且可产出达到零级产品要求的 Li2CO3。全湿法的流程为碱溶解、酸浸出、P204 萃取净化、P507 萃取分离钴和锂、H2SO4 反萃回收 CoSO4 和萃余液沉积回收 Li2CO3 等六步。

碱溶解的目的是去除并回收铝。原料中的铝以金属形态存在，用 10％ NaOH 溶液溶解正极废料即可使两性金属铝以 NaAlO2 的形式进入溶液，同时覆盖在铝箔上的疏松钴酸锂膜可以加速铝与 NaOH 的反应。LiCoO2 不溶于碱，全部进入碱浸渣，而 NaAlO2 全部进入碱浸液。此时用 H2SO4 中和 NaAlO2 溶液即可获得 Al (OH) 3 沉淀。

酸浸出和 P204 萃取净化过程可以使铁和铜等杂质与含镍、锂和钴的物质分离。以固液比约为 1:8～1:10 配制浸出液，在使用 2.0mol／L H2SO4、3mL／g H2O2 溶液，于 80℃浸出 90min 时，锂和钴就能完全进入酸浸出液。过滤后，调整浸出液 pH 值，用 P204 萃取除杂。完成后 Al、Fe、Zn、Mn、Ca 等杂质进入有机相Co、Li、Ni 等仍留在水相，便于后期处理。该步骤发生的反应为

2LiCoO2＋3H2SO4＋H2O2 → 2CoSO4＋O2↑＋Li2SO4＋4H2O

酸浸出后使用 P507 萃取分离钴和锂。控制 pH 为 5.5，P507 浓度 1.0mol/L，使用煤油作为稀释剂，皂化率为 75％ 时，可基本使锂和钴萃取完成。共萃入有机相的少量锂可用 CoSO4＋H2SO4 溶液洗涤除去。得到的反萃取液富含 CoSO4，可用于电池生产。
溶剂萃取回收钴后的萃余液经浓缩后，加入饱和 Na2CO3 溶液使锂以 Li2CO3 的形式沉淀。滤饼用 95°C 左右的热水洗涤两次后，除去结晶母液和 SO42-，再烘干，即可获得零级产品要求的 Li2CO3。

# 2.1.4 溶剂萃取法 [5]

该工艺由南俊民等人提出，可简化酸碱浸出法，并且使操作更简便。该工艺包含废旧电池预处理，Acorga M5640 萃取回收铜，Cyanex272 萃取回收钴三步。

论文特别阐述了废旧电池在进行化学处理之前的操作。由于废旧电池往往带有剩余电量，所以在流程开始前需要放出额外电量。把这些单体电池放入一个有水和电子导电剂的钢制容器中通过使单体电池的正负电极短路而使电池完全放电。用专用机械设备把完全放电的单体电池外壳打开，并用磁选法分离出铁外壳。为防止电池内的 LiPF6 有机溶液和水反应生成 HF，应立即进行碱浸，同时可将铝浸出，利于在下一步生成 Al (OH) 3 沉淀。此时发生反应

LiPF6＋H2O → LiF＋POF3＋2HF
HF＋NaOH → NaF＋H2O
2Al＋2NaOH＋2H2O → 2NaAlO2＋3H2↑

在室温下按固液比 1:10 反应 5h 以后即可浸出绝大部分的铝，而钴几乎无损失。为满足后面对钴和铜的萃取分离要求，研究者选取了 H2SO4 溶解碱浸后的剩余物质，并且实验表明：在室温下用 4mol/L H2SO4+H2O2 溶液 ［ n（H2SO4）∶n（H2O2）＝3∶1］溶解废料在固液比为 1:10 反应时间大于 2h 后钴的溶出率即可达到约 97％。值得注意的是，H2O2 可加速溶解过程。该步骤发生的反应有

2NaAlO2＋H2SO4＋2H2O＝2Al(OH)3↓＋Na2SO4
2MCoO2＋3H2SO4＋H2O2＝ M2SO4＋2CoSO4＋O2↑（M 为 Li + 或 Na+）

分离出铁、铝杂质之后，可开始萃取铜。Acorga M5640 萃取剂中的肟结构对铜具有很高的选择性，这使得 Acorga M5640 可在一定的酸度条件下成为铜的特效萃取剂。室温下使用萃取剂浓度为 10%，萃取液 pH 值为 1.0，油水相比为 1:1，萃取时间 10min，同时在室温下使用 2mol／L 的 H2SO4 洗脱，按相比 1:1 洗脱两次平衡时间为震荡 1min，可以使铜的回收率可达 98％。

选取浓度为 1mol/L 的 Cyanex272 从萃取铜剩余液中萃取钴。其方法和萃取铜的操作类似。使用 1mol/L 的 Cyanex272 萃取钴的适宜条件是在室温下、萃取剂的皂化率为 10％、溶液的 pH 值 5∙5、相比为 1:1、反应时间为振荡 1min 时。反萃取则是室温下用 2mol/L 的 H2SO4 洗脱、相比为 1:1、震荡时间 1min、洗脱两次。钴回收率可达 97％。

回收铜、钴后的萃余液，通过加入 Na2CO3 的方法使锂沉淀下来回收，得到 Li2CO3 产品。

# 2.2 磷酸铁锂 [LiFePO4] 电池 [3]

近年来，磷酸铁锂电池发展迅速。磷酸铁锂具有环保、比容高、价格低廉等特性，因此逐渐被重视。对于磷酸铁锂电池的回收，吴越等人的湿法冶金法可起到积极意义。该工艺由碱溶解、H2SO4 浸出和铁的分离、HNO3 浸出和铁的分离以及锂的分离等四步组成。

碱溶解过程的目的是回收铝。使用 5% 的 NaOH 溶液溶解正极材料可使铝箔在 30min 内完全溶解。此后再调节 pH 至 9.0 使铝以 Al (OH) 3 的形式沉淀下来，并且其品质可达到化学试剂标准。H2SO4 浸出步骤可以使滤渣中的铁和锂以离子的形式进入溶液。以
2 mol/L H2SO4 和 30% H2O2 的浓度及固液比为 1:10 的条件在 60 ℃下搅拌 2 h，过滤后用 NH3・H2O 和 5% NaOH 溶液 (体积比 1∶3) 调节 pH 值，可使 Fe3+ 以 Fe (OH) 3 沉淀下来。需要注意的是，该步骤中会有少量 LiOH 沉淀。沉淀物经灼烧即可获得高纯度 Fe2O3。

硝酸浸出工序能使磷酸铁沉淀，利用的是 FePO4 微溶于 HNO3 的原理。FePO4 沉淀后，Li + 会单独存在于溶液中。H+ 浓度、双氧水的含量、搅拌时间和搅拌的温度都对此过程有影响，经实验可得最佳条件为 HNO3 浓度为 3 mol/L，双氧水的含量与固体滤渣比例为 3 mL 30% H2O2 /1g 固体滤渣。过滤后的滤渣仍然存在炭黑等杂质，加入 H2SO4 可以让磷酸铁溶解，再用 NaOH 使铁以 Fe (OH) 3 的形式沉淀。
分离锂需要将锂转化为 Li2CO3。需要向滤液中加入饱和 Na2CO3 溶液，并且控制温度为 90°C。滤饼用热水洗涤两次后烘干。热法沉淀 Li2CO3 可达零级产品要求，一次沉锂率为 80% 以上。

# 2.3 铅电池 [4]

铅电池在生活中应用广泛，尤其是汽车内的铅酸蓄电池。铅电池的主要回收流程为解体、分类和再生。

解体步骤中，将 H2SO4 放出后单独回收，机壳用破碎机解体，用比重法 选出塑料后，再分为极板、极柱、电池槽和盖等。分类时将除去塑料的含铅部件破碎成 60mm 以下的小块后分为 4 类：铅粉，占总量的 64%，其中含铅 70%；铅泥，占 5%，其中含铅约 3%；小块铅合金，占 7%，其中含铅 26.5%；铅渣，占 14%，其中含铅约 0.5%。再生步骤有三种：第一种是将上述铅粉和制铅厂的烟尘一并处理，制成含锑 1.7%-1.9% 的的电池用软铅再生利用；第二种是将铅泥供转炉处理；第三种将小块铅合金作金属配料。

# 2.4 镍镉电池 [4]

镉是电池、颜料、塑料稳定剂、电镀和合金等方面的稀有金属，因此从废旧电池中回收镉也十分有益。国际上成熟的镍镉电池处理方法有两种：干法和湿法。
干法主要利用镉及其氧化物蒸气气压较高的特点和镍分离。一种工艺流程为先剥离电池表面覆盖物，然后在 900-1200°C 下进行氧化焙烧，使之分离为镍烧渣和 CdO 浓缩液：镍烧渣作为钢铁冶炼原料使用；CdO 浓缩液经浸出净化后制成镉盐。

# 2.5 锌锰干电池 [4]

锌锰干电池是日常生活中使用最广泛的电池之一，且由于锌锰干电池是一次电池，报废量大，因此对锌锰干电池回收的研究也十分完善。
北京科技大学的研究者们采用湿法工艺流程，能充分对干电池进行无害化、资源化处理，提高了废旧电池的利用率。此方法的锌的总回收率为 83%，含汞废渣可送专门工厂处理，工业废水处理后可循环利用，减少二次污染。
中南工业大学蔡传算、黄茂春等人发明了从废锌锰干电池中提取二氧化锰及锌的方法。该发明是将废电池破碎后分选出的 MnO2 物料与金属锌渣或 Zn2S 精矿在 H2SO4 溶液中同时浸出，同时用铁强化浸出，提高了 MnO2 和锌的浸出率，并省去了常规方法的焙烧工序，提高了金属回收率，降低成本。
李寅高发明了一种干电池厂含氨的锌废水处理方法，处理线由贮液池、沉淀池、萃取器、除油器和中和池组成。贮液中的废水泵入带搅拌的沉淀槽中，边搅拌边加入氨水，使生成 Zn (OH) 2 渣，静止，放清液陈化，沉淀槽中下沉的渣用离心机分离，清液用（C6-C18）混合伯胺有机萃取剂萃取，氨水反萃取，盐酸再生。

# 3. 生物法处理废旧电池 [6]

除了通过化学反应提取需要回收的物质外，借助微生物的生化反应也是可行的措施。生物法可以避免化学法的二次污染，同时也可处理有机废物。杜竹玮团队发现了一种能还原锰的细菌，可以辅助废旧电池回收。
海底沉积物中的嗜酸混合异养菌株是本团队的研究对象，该菌种可直接和间接还原锰。直接还原过程是锰在电子转移体作用下被还原，此时的电子转移体是有机物中的碳。该反应的方程式为

24Mn+C12H22O11+48H+→24Mn212CO2↑+35H2O
Mn2CO2+H2O→MnCO3+2H+

间接还原是微生物分泌有反应活性的小分子如有机酸、硫醇、醌、酚以及其他具有还原性的无机代谢物与四价锰反应，或通 过 改 变 环 境 的 pH 等促进反应进行。
锰还原 异 养 微 生 物 利 用 废 旧 电 池 粉 末 中 Mn (Ⅳ) 作为代谢呼吸链最终电子受体，传递氧化有机物产生的电 子，消 耗 代 谢 产 生 的 酸，浸 出 回 收 金 属锰。驯化后微生物对重金属的耐受力提高，培养基中电池粉末含量低于 3% 时，浸出率在 90% 以上。

# 4. 结语

目前对于废旧电池回收的研究已经日趋完善，工艺也在不断完善：从粗放的填埋式，到酸碱浸出法，再到生物法。在电池回收技术发展的基础上，我们也需注意电池以外的角度。例如，厂商应研发本身就利于回收处理的或寿命长的电池，以从源头缓解废旧电池问题，并提高资源的利用率。与此同时，提高人们对废旧电池回收的积极性也尤为重要，特别是通过政策等引导，使废旧电池基本能得到回收。无论如何，废旧电池的回收依然有光明的发展前景。
