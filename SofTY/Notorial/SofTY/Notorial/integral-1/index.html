# 函数的极限

# 有限点处的极限

有一个函数，如果在自变量 x 无限接近某个确定的值 x0 时（记为 x→x0，x≠x0），函数值 f (x) 无限趋近于确定的常数 A，就可以说

limx→x0f(x)

幂函数、指数函数、对数函数等初级基本函数在定义域内每一点处的极限相当于此处的函数值。

# 无限大处的极限

有一个函数，如果在自变量 x 无限增大时（记为 x→x0，x≠x0），函数值 f (x) 无限趋近于确定的常数 A，就可以说
limx→∞f(x)

# 函数极限的运算法则

lim[f(x)±g(x)]=limf(x)±limg(x)

lim[f(x)⋅g(x)]=limf(x)⋅limg(x)

limf(x)g(x)=limf(x)limg(x)

# 无穷小

# 定义

当 x→x0 时或 x→∞ 时，f (x) → 0，则这时 f (x) 叫做 x→x0 时的无穷小。即

limx→x0f(x) = 0
时的函数值

# 无穷小的比较

如果 limx→x0α=0limx→x0β=0

若limx→x0βα=0，那么β是比α高阶的无穷小 若limx→x0βα=∞，那么β是比α高阶的无穷小 若limx→x0βα=C≠0，那么β是比α同阶的无穷小 若limx→x0βα=1，那么β是比α等阶的无穷小，记为α∼β

常用的近似：
当x→0时x∼sin⁡x∼tan⁡x∼sin−1⁡x∼tan−1⁡x

# 等阶无穷小的变换

如果α∼α′,β∼β′,∃limβ′β,则limβα=limβ′α′
