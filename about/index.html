# 本站

本站是一个普普通通的博客网站，基于 hexo。目前上线运营的版本为 Cyberpark Nichome，最开始的版本为 Cyberpark Prime（已封存），另一个短暂上线于 2024 年底并很快退出运营的版本是 Cyberpark Trinity（已设置为不公开），在 2027 年前后本站将迭代至 Cyberpark Tetra。

选择 hexo 以及使用 github pages 默认域名的目的也很简单 ——STY 没钱买服务器和域名。当本站目前的 github 仓库满了的时候，我就会动用其他的电子邮箱地址多注册几个账号并建立其他的仓库，绕开 github pages 一个账号只有一个 github pages 仓库的限制。对于域名问题， 我早就发现冷门域名可便宜了， .online 域名最低能干到十年￥51（2021 年数据），快跟免费的一样，再等等估计就能入手啦。

关于本站的内容，我的定位是 “自己的互联网后花园”。顾名思义，主要是有关学习生活的奇闻轶事和琐碎的感受。

与此同时，我们也有一个 chatbox，事实上是我的朋友圈，里面有大量逆天视频和抽象发电语录。

本站名称的来源则比较复杂。 Ameitsu 是典型的日语罗马字。其中 Ame 明显来自雨（あめ），但在这里其实是指天，あめ其实可以是天的训读。 itsu 是某个只有汉字音读的日语汉字，基本只在一个欧洲国名的日语汉字名中使用，现在这种写法已经不是正式的了。（看你能不能猜到 (o´▽`o)）。 Ameitsu 和 STY 真名有关系。 Cyberpark 是 “Cyberpunk' 的变体，但和后者没关系。

本站有一个站娘，名字叫做 Clare Logan Halifax，也是本站的守护天使。她的昵称是小盐，因为她的名字各部分首字母为 C，H 和 L，看着跟盐酸似的，但直接叫盐酸太绷不住了，因此就取一个 “盐”。还挺可爱的。

本站有一个背景故事，如下：

本站还运用了世界领先的生化技术。本站除了存储和部署模块的其他文件全部位于一只猫娘的大脑中，透过目前最先进的脑机接口协议与 STY 的电脑相连。关于那只猫娘，她自称叫■■■■，应该不来自现实世界，她在■■市■■区■街道■■■号的■■外国语学校外被发现，当时她奄奄一息，可能刚进入现实世界。后来在 2022 年 4 月 31 日一个匿名的人外研究者发现了她，对她进行了基本的救治，但发现她的健康在正常情况下迅速恶化，只有在大脑与芯片连接并且身体浸在■水合■■化■■■代■■酸，一种先进的医疗用化合物中才能生存。在安装脑机接口后，那名人外研究者试图在维持猫娘正常意识的情况下植入微型处理器，将她改造成活体服务器。以上内容均为那个人外研究者直接跟我直接说的，STY 不能保证其准确性。但是 STY 保证她不会受到任何伤害，确保本站的内容不会使她的大脑超负荷运行。目前该猫娘每天大约有 37.8% 的时间处于意识湮灭状态，在另外 20.3% 时间能试图通过脑机接口和外界进行交流，但目前无法从回应中推测出她的出身和原本的世界

作为有思考能力的人类个体，你应该知道世界上并没有猫娘，即使你逼着你家猫看猫娘插画也不能让它变身。因此，以上内容纯属虚构，如有雷同纯属抄袭。

本站目前主打的专栏是 @STY 系列，该专栏围绕 STY 的学习生活和学校活动展开，尤其是其子专栏 Cyberpark Weekly ，实质上是 STY 的周记。除此以外，STY 的朋友九万里偶尔会有投稿，虽然经常是我主动求她。鉴于 STY 目前处于高三阶段，本站剩余的专栏基本处于停摆状态，诸如：

# STY

以下内容均为现实时空。

我就是 STY，STY 就是我。

STY 是北京市朝阳区的一名高中生，就读于某一类校。性别为 ♂ ，但不止一次有人问过 “STY 你为什么不是女的啊 ？” 这种问题。（因为我比较可爱） 现在已经成年啦！

STY 是我名字的拼音首字母，Ameitsu 是我的常见网名，但在行文中我一般只使用 STY。

介绍自己不能 “自卖自夸”，我们需要多元的角度。根据周围同学的评价，STY 是死傲娇、技术宅、猫娘和抽象哥们。

在技术方面，STY 主攻 Python 和 C#，正在学习 Unity 和 C++（给竞赛做准备），同时也零零散散地接触过一些语言或技术，例如 web 三件套 HTML、CSS 和 JavaScript, Markdown, JSON，xml，yaml，网络爬虫，数据库等（你应该看得出有几个凑数的）。事实上这个网站纯属意外，它最开始的目的是往 github 上刷功德（贡献值）用的。

STY 在宅文化方面比较矜持，可能是因为尴尬症容易发作的缘故。STY 偶尔会追番，但看片的进度实在是太荡气回肠了。STY 曾经一集 GBC 看了整整一个 2 月，有点太佛系了。流行的二游咱一点不碰，音游也是，咱上次打开音游还是在 2022 年。不过 STY 玩的主要是 itch.io 上各种二次元独立游戏，我偶尔会在 Cyberpark 上聊一聊。STY 听 VOCALOID 非常多，但不太想发表太多评价。

STY 比较喜欢学习语言。目前学习的外语有英语、日语和西班牙语，其中英语水平能横扫整个朝阳区（区排第二），西班牙语也到了能和西班牙小朋友唠一唠的水平（CEFR 体系的 A2）。STY 也接触过拉丁语、希腊语和因纽特语，但这些就纯属娱乐了。

STY 是铁路宅，但没什么运转的经验，知道的铁路知识也没那么多。STY 关注的范围比较广，从国内、西欧、美加乃至非洲咱们都清楚一点。另外一个 STY 非常喜欢的是星球大战，你可以从本站随处可见的星球大战台词烂梗看出来。

STY 是互联网社恐，暂时只提供电子邮箱地址，可以凑活聊聊。

对了，我们不得不再提一句九万里老师。她是我的同学，自称双性恋，但大家都以为她是女同，而且事实证明她真是。她相当多才多艺，从生物学、物理学、民族乐器、国画乃至西方哲学都有精通的地方。STY 和九万里一直维持着传统友谊，有时候会在线下贴贴，她还为本站友情供稿了几篇。STY 承认过自己喜欢她。

希望你在本站看得开心ヽ (・∀・)ﾉ

Best wishes,
STY
