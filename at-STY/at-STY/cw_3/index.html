# STY has sth 2 say

欢迎来到本学期第四周的结算环节！一看日历，竟然已经快一个月过去了！所以说能不能把新手保护期再延长一点？

话说回来，本周有一个鲜明的关键词：“塞翁失马，焉知非福”。这周，或者说这一段时间里我碰上了一些意料之外的事情，但最后无论如何咱们都想出了解决方案，抑或是实际上这些事完全用不着担心。其实，很多事完全没必要焦虑，只需要顺势而为，用在线的智商去解决问题，就好啦！比如说 STY 作为班长替全班给学校打工的时候，不用担心 deadline 和人选的问题，要相信自己 “烂摊子处理大师” 的信誉。

还有一件小事儿，那就是我忘了给 Cyberpark Weekly 系列编号了！ Cyberpark Weekly 是 @STY 的子系列，同时有两个编号：在 Cyberpark Weekly 里是 vol.n，在 @STY 里是 vol.(n+23)。以本篇为例，其编号是 Cyberprark Weekly vol.3 和 @STY vol.26。别担心，cyberpark 的编号系统是一个玄学，先不提 Ameitsu Orchestra 神秘消失的编号，最近的还有 @STY 的超级大跳号，从十几直接蹦到二十几，虽然有 Cyberpark Trinity 的影响在，但的确有近十个编号从未存在过。

那我们开始啦～

# 学雷锋日历经奇遇，志愿活动 STY 再出发

众所周知，3 月是学雷锋月。到了这个时候，学校肯定会组织学雷锋班会。但从第一步上就有问题了：为什么非得班会前一天才通知我做 PPT 捏？没事儿，咱有超强的 ppt 快速制造功能。

这次的班会更多是要评出各班的学雷锋榜样，最后的投票结果是 STY 和女班长并列第一。第二天，班主任说由于名额有限，咱只能挑一个人报上去，所以要求我们俩各写一个和雷锋精神有关系的材料。真棒！特地挑个作业最多的周一布置这个活儿！

只能写呗。我想写点关于志愿者精神的事儿，毕竟它和雷锋精神的联系很紧密，况且咱还是志愿北京的注册志愿者嘛！让我想想咱都干了点啥 ——

  • 志愿总时长 215 小时
  • 参与 49 个项目
  • 全北京排名 658758
  • 被学校多次评上 “劳动奉献” 称号

被评上 “劳动风险” 主要是合格考那天我被晾在零下十五度的八十中门口给大家签到。

虽然确实从量上不是很多，但 STY 喜欢这种给 greater good 打工的感觉。

# 图床再挂无需紧张，转变思路才是良方

是的，我们的图床第八次挂了。(*ﾉωﾉ)

这次的原因很简单：我白嫖我提前存在 notion 在线页面上的图片链接，没想到上面的图片居然是实时刷新的，而且每次的失效时间都是不一样的！然后就没辙了。

不过我们还有解决方案，全都脱胎于我之前想过的馊主意：其一，注册一个只有图片的微信公众号；其二，用我的各种马甲电子邮箱注册多个 github 账号，再新建一些 GitHub pages 来存图片。

当然了，STY 认为最合适的做法就是 —— 图文分离！我可以指定一个 notion 页面，把图片进行编号和解说，再在正文里贴上这个页面的链接和图片的编号！逃避可耻但有用嘛。

# 抓壮丁 STY 误打误撞当裁判，办球赛朋友们或多或少心忐忑

我们学校要举办女子排球赛。每个班出十个人当队员，三个人当裁判。由于剩下的一个裁判没人当，所以班主任就把我供出去了。问题来了 —— 我不会排球规则啊！之前打排球都是听别人说的得分情况。好在三个人里有两个只用盯着出没出界的边裁，于是咱就当了边裁。比赛尚未开始，具体的体验估计要到下周的 Cyberpark Weekly 里再说了。

但是这排球赛的讨论度超级高。再往后我们还有男子篮球赛，和这次的女子排球赛一起，带来了一个现实的问题：咱们班没有排球和篮球打得好的同学！参赛队员都声称这次我们班将喜提倒数第一…… 事实上我们班真正的强项是羽毛球，年级最顶尖的一批选手都在我们班。现在你知道偏科是什么舒爽的感觉了吧！(Φ ω Φ)

# 选课未果却非不幸，柳暗花明还有路行

上次咱不是选课挂了嘛。最后经过二次选课，咱们最后被分到了物理选修。
其实这个结果也不错捏。而且咱赶上好时代了，从本学期开始物理选修开始讲和校内关系更深的内容，比如说高考和模拟题的压轴题。所以说前俩学期他们都干了点啥，才让一些同学去投诉？

第一节课讲的是动生电动势和感生电动势的本质，也就是围绕着那个天杀的破导轨和杆子说事儿。动生电动势的例题问杆里电子受到的洛伦兹力在能量转化中起到了什么样的作用，听完后我才反应过来电子受的平行于杆的洛伦兹力负责消耗电能，垂直于杆的洛伦兹力负责将电能转化为动能。STY 做物理卷子的最后一题经常有这种无力感 —— 我™在写点啥啊！？就不能写出点 “优雅” 的话吗？后面的感生电动势也挺神奇的，比如说感生电场里莫得电势，但有电势差，所以感生电动势里产生的只是电动势。无论如何，感生电动势和动生电动势最大的特点是一个典型模型使劲考，所以咱在这儿听懂一遍后如果再看着类似的题就相当于情景默写了。

# 风吹朝阳初春云，雪满北京三月天

是的，在 2025 年 3 月 15 日，北京迎来了 2025 年的第一场雪。

其实北京在 3 月份下雪不是什么奇怪的事情，在 2022 年 3 月北京也下了场大的，当时天边呈现诡异的粉色。真正奇怪的是北京一个冬天都没正经下过大一点的雪，几乎只是在山区小打小闹飘点雪花。

这次雪的规模也不小，虽然前前后后只有三个小时，但雪花飘落的速度和雪花的大小真的让人记忆犹新，完全达到了 “未若柳絮因风起” 的地步。当然了，下雪之后朋友圈里的实拍图就嗖嗖地全刷新出来了。幸亏现在是 3 月，地面的温度较高，雪根本积不起来，也不会对交通产生影响。

就酱！！！
