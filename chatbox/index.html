突然有了一个用sympy搞出来的小点子。一个半自动的解析几何计算器！还是STY的经典才艺——先进的命令行技术！大概聊一下吧。

项目名为 Ameitsu Coniseca，简称是……额……CS吧。不能和伟大的AC重合。

主程序启动后，就是一个命令行输入界面。输入“new”即可进入计算流程，该命令会在本地生成一个以时间戳为名称的文件夹，里面有 exp.cnsc、dots.cnsc和values.cnsc三个文件，以上三个文件分别存储直线或椭圆方程、点坐标和中间值。

setEllipse(a = "",b = "")会新建一个椭圆方程。设直线的指令有两种，一种是纵截式的，即primeLine(k = "",dot = "",default = "",default2 = "")，另一种横截式是primeLine(t = "", dot = "",default1 = "",default2 = "")。若k或t留空，则会自动设置为k或t，也可以从values.cnsc中提取值带入。两个以default为名的是该直线与椭圆的交点，不指定则直接设为(x1,y1)和(x2,y2)，也可引入一个已知点。

将点坐标存入计算流程的方式很简单，就是SetPoint(name = "", x= "",y = "")。求斜率也是，只需输入slope(point1,point2)即可，同步会存入计算流程。

接下来是创建直线。创建直线和设直线不一样，设直线会引入未知数，创建直线是利用未知数。输入setLine(point1,point2,k,v)即可生成。若两个point均存在，则利用它们计算斜率，带入点斜式方程；若存在k和1个point，则直接带入点斜式方程。若还有v标记，则自动将斜率变为负倒数。

为直线方程赋值的操作如下。输入solve(line,x="",y="")，若单独存在x或y则带入进行运算，同时存在即报错。结果会以点形式存储。

联立的操作是combine(exp1,exp2)，可以是椭圆方程和直线方程之间，也可以是直线方程之间。如果是前者，则会输出一元二次方程，自动求解判别式并展示韦达定理结果；如果是后者，则会返回交点坐标。

还会有一些其他的功能，如验证三点共线（相当于验证两个斜率相等）和验证点是否在椭圆上。

远期还可以有解题过程展示功能。会有一个操作存储器，当用户选择输出解题过程时，会展示每一步的序号和操作的任务（如“1 - 设椭圆方程，参数为a=2，b=1”），用户按需要输入序号，程序读取后重新计算并输出。
