# 建 站 宣 言 ！

# 2023 年 1 月 21 日，Ameitsu Cyberpark 正式开始运作

期间，为了达成在人类互联网世界有立足之地，STY，克服了以下困难

  • ssh 连不上
  • ssh 连不上 github 的 22 号端口
  • githubpages 被墙
  • bash 出现极大的问题
  • ……

但是，不论如何，Cyberpark 还是开放了！
现规定 Cyberpark 的内容

  1. 插画 (全年龄 doge)
  2. 技术问题
  3. 睿智梗图
  4. 离谱游记
  5. 数理化离谱题（远期计划）

编辑于 2023/1/21，春节前，STY

# 更改内容

  1. 删除了 原文中直接的第一人称
  2. 删除了 内容一节中梗图的说明
  3. 通过了 在单独页面中编辑 Cyberpark 章程的决定
  4. 增加了 新章程删除条目的要求

# 内容说明

  1. 建站宣言是较重要的页面，不应使用第一人称
  2. 梗图的获取方式为从视频中截图，然而操作不易，难以一次更新多张图片；尽可能减少一些空间损耗
  3. 独立的页面是重要页面的基本要求
  4. 目前已有删除段落的操作，需要统一的规定以确保格式一致
  5. github pages 的空间为 1GB，在若干年后会耗尽
