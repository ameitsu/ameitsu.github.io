谁能想到即使是在工作日里，故宫人都多得和平常一样。对了，这也是 STY 在国内见到外国人最多的一天。

五一一回来，学校就组织了故宫研学。但是只要一说到 “研学” 这俩字儿，STY 就绷不住了，毕竟 ** 上一回的研学咱也没更完呢！** 所以咱们就一篇结束，别太长，写的有意思点儿。

这段时间里，STY 一下成为了我们班班长 + 支书 + 英语课代表，最不缺的就是活儿，所以这一篇也被鸽到了现在。

# 众所周知，从北苑到奥运村和从奥运村到皇城根用时一样。

如题，STY 所乘坐的学校大巴车一出校门就被堵住了。我寻思已经过了早八了为什么车还这么多呢？

我一看：哦，原来是因为这里是北京，没有时候不堵的。北京的道路交通情况简直太棒了，堵车根本不存在，谁需要地铁呢，坐车才更快嘛！哎呀，北京的道路交通情况太棒了，堵车简直是个美妙的体验啊！每天都能够享受到堵车带来的愉悦和放松，真是让人感到幸福至极啊！车子飞驰！北城的快速路简直难以数计，这简直太让我愉悦了！北京老城区的街道上挤满了汽车，宛如一条巨龙纠缠在狭窄的巷道间，让人感到窒息！每当我等在红绿灯前，那漫长的等待时间仿佛就是在嘲笑我的无助和焦躁！愤怒的车笛声不断响起，仿佛在诉说着每个人内心的不安和急迫！北京老城区的车太多了！路太窄了！红绿灯时间太长了！让我感受到了生活的挤压和无奈，但也让我更加热爱这座城市，因为这里是我的家！

晃荡上了安立路之后，接下来就是使劲往南，直到一个位于美术馆后街的 dead end。大巴车从充满本世纪初气息的亚运村门口穿过，在安慧桥上多花了点耐心跟在已有六十多岁的 108/124 路无轨电车身后；目光从上世纪八十年代的商住混合筒子楼，转向十二号线的地铁工地；心里想着的，是下一座立交桥叫什么，要穿过的下一条环路编号是多少……

到地坛牌楼后，两侧的居民楼逐渐退去，因为古今建筑高度鲜明对比而形成的空旷出现在面前。到这里，你就知道已经抵达北京老城区了。接下来就是沿着安定门内大街一个胡同一个胡同地向南走，当北京中医医院的粉色主楼出现在眼前时，当中国美术馆的金色琉璃屋檐的反光两次映入眼帘，你就能知道……

这大巴车好像没地方停啊！！！！！！

# 捎带手的北京胡同游

果然，STY 知道大巴车绝对不可能在故宫附近停靠，毕竟路边都有栏杆。大巴车兜兜转转在北河沿大街停了车，接下来的路就要我们走了。

一转过弯，东华门夸嚓一声闪亮登场。这条位于东华门前方几百米的小街上集合了老北京相当一部分的商业种类……

  • 外地游客面向型经济：老北京烤鸭贩卖业，清朝服饰租赁业，瓶装豆汁儿零售业，北京文化衫零售业，老酸奶和冰棍零售业……
  • 东西城区面向型经济：马扎零售业，非老字号餐饮业……
  • 其他区北京人面向型经济：麦麦，免费指路指厕所服务……

    在东华门前等候时，STY 见到一批又一批的外国游客。不过一想也是，外国人来北京玩肯定得巧妙地避开双休日和中国法定节假日。在整个游览过程中，STY “窃听” 了很多外国盆友的对话，居然还碰上说西语的，这是 STY 学半年西语以来第一次在现实世界中听见西语对话，给孩子感动坏了。

故宫护城河里的鸭子啊！它们那双明亮的眼睛，那柔软的羽毛，简直让我心神荡漾！我忍不住想，一口把它们全都吃掉，尽情享受它们那鲜美的肉质！啊，这种冲动真是让我怦然心动，欲罢不能！不，我要抵制这种欲望，这些可爱的鸭子不该成为我的口中美味！但是，但是…… 我真的好想吃啊！啊啊啊！我要控制住自己，不让这肆虐的食欲控制我！这些可爱的鸭子，它们应该被保护，而不是被我满足口腹之欲！我要忍住，我要忍住！但是，但是…… 啊！这鸭肉的香味简直让我抵抗不住！不，我不能！我不可以！啊啊啊！救命啊！我太想吃了！快来救救我！我已经快要被这些可爱的鸭子征服了！啊啊啊！

以上纯属发电。

# 不知道第几次沿故宫中轴线转悠

进了故宫，意料之中，我们的路线是顺着故宫的中轴线从南走到北。从午门进去，穿过太和门，来到三大殿前面；看一下太和殿里皇帝的宝贝椅子，瞟一眼中和殿里一片昏暗的陈设，瞅一瞅保和殿的牌匾；如果预约过，就去珍宝馆转转；如果没预约过，那就去御花园溜达一圈。这是最普通的故宫游览路线了，STY 觉得我们好歹作为北京当地人，自己去故宫都三四次了，说不定能来点新花样。您猜怎么着？没变。

不仅我们要走这个无聊的路线，我们还要完成一个实践手册。说是 “手册” 都抬举它了，因为它只有四页，里面内容完全能拼到一张正反面再生纸卷子上。里面的题则是一些动动小手一搜就能出的那种，有人在大巴车上都开挂写完了。不过还是知足吧！还有别的学校组织了故宫研学，他们的研学手册那叫一个厚……

嘿，朋友们！我们不仅要一起走这条微风拂面的路线，还要完成一个超级实用的手册！当然啦，这个手册说是 “手册” 有点吹嘘了，就四页纸而已，内容简单到可以写在一张纸上。里面的问题简单到动动小手就可以解答，有些人甚至在公交车上就能完成！不过，要知足哦！其他学校组织的故宫研学手册那才叫一个厚呢…… 不过，我们还是要相信自己可以轻松应对，一起加油吧！🌟📚💪

真正有意思的，还是自由活动时间。

# 故宫里野餐，讲究！

解散之后是午餐时间。想找个伴也不可能了，因为全年级 240 个人在 750 万平方米的故宫中扩散的和学校里劣质香水的扩散速度有一拼。

STY 中午吃的是金枪鱼鸡蛋 sandwich，金枪鱼饭团和一些可能是临期食品的东西（至少我有一阵子没见过他们了）。进餐的地点也挺随便，就是皇极殿的地基上。

吃完了，有人问 STY 是高中的还是大学的。首先，STY 没听说过谁家大学生天天穿校服出去的。

# 一些必要的小消费

STY 吃完饭就往西去了开在军机处对面的文创店。然后花￥98 买了故宫猫猫摆件，故宫猫猫便签夹和故宫猫猫笔记本。开箱：




# 猫猫！

这次是 STY 第一次在故宫看见猫猫，但很可惜就留下一张特别糊的照片。

在军机处历史书圣地巡礼转一圈后，STY 沿西侧的通道向南走，想看看这片充满办公区的地方藏着多少猫猫。不过看到的小学生数量远大于猫猫数量 —— 二者比例约为 400:1。

沿途，STY 路过了著名的故宫餐厅和冰窖咖啡厅：这二位的共同特点是随便两三个菜的重量远小于我刚买的三件文创产品，但价格却一样。然而居然还有不少我校同学在里面团建 —— 我对各位的经济情况有了更深的体会。

一路走到武英殿，靠在内金水河的汉白玉栏杆上时，STY 看见了猫猫：

很好奇这只猫猫在健身区里干嘛。可能是刚做完引体向上？

# 瞎逛

然后 STY 开始了故宫的瞎转之旅。全天的行程计算下来，STY 绕着故宫转了至少两个整圈。

那天的气温简直就是烧火烧到我要爆炸了！我要把太阳挖出来掐死！但是突然发现小卖部里的蜜桃乌龙茶竟然才卖 6 块钱！我当场惊呆了！这简直就是上天给我的馈赠！我觉得自己要被幸福冲破天际，我要唱歌跳舞，把整个世界都告诉它我有蜜桃乌龙茶喝！哈哈哈哈！太开心了！太有人生意义了！我的灵魂终于得到了拯救！太棒了！就是太棒了！

从武英殿到文渊阁；从保卫部到文物部；从四星级厕所到员工篮球场；从 “地上天宫” 到 “故宫礼物”……STY 在不知不觉间把曾经没有探索过的，因为不甚有趣而鲜少有人提及的地点领略了一遍。要不是这次漫无目的的散步，STY 不会知道东华门旁边有猫猫据点，不会知道穿过一座又一座宫门时就像进入超空间一样，不知道坐在树荫下红墙边的长椅上是多么惬意……


也许，这就是 游记 系列的核心了⑧。从后奥运时代的新首钢，到徽州古城，再到荒无人烟的清河营郊野公园…… 贯穿本系列的线索就是探索喧嚣背后的惬意。虽然这明显是 retcon，但 STY 认为这就是这个系列的最高目标。

好了，不要被热死了，快点跑回御花园里吧！（不，还是回家更爽）



就酱。
