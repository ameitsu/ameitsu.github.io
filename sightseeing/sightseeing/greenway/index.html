# 所以说我们有了很多绿道

这年头，北京突然冒出一些叫做 “绿道” 的东西。它们无非有着这些特点：

  • 长度：动辄数十公里，搞得比某些快速路都长
  • 选线：就是把一堆公园和自行车道连起来了
  • 宽度：双向半车道。是的，拼起来才是一条车道

在这些已经完工的绿道中，有三条气质稍微特殊一点的，而 STY 有幸就住在它们中间的■运村。让 STY 发挥一贯的锐评技巧来评价一下：

  • 朝阳绿道：假如你想欣赏一下东四环的绿化带和安立路的辅路停车场，那么这就是你最佳的选择。
  • 昌平绿道：这是一条以标志不清晰和柏油路质量堪比未铺装公路的自行车道，而且沿途的补给点多的堪比北四环上的加油站。
  • 自行车专用路：骑车时最憋屈的事情之一，就是前方几百米的直道上空无一车，而你只能压着十五公里每小时的最高限速往前爬。在这里你就能体验到。

这次体验的昌平绿道，主打的是里程长和惊险刺激：规划中的路线全长四十三千米，但是宣传上写的都是四十二前面，真的是太谦虚了，我哭死；惊险刺激，是因为没有任何人提醒你沿途有多少工地和需要自己个儿绕行的路线具体怎么走。

虽然昌平绿道的标志相当不清晰，路面质量堪忧，甚至没有百分之百确定的路线，但这就是它的乐趣所在：我们管它叫城市探索。Citywalk 本质上只是人文气息浓厚的遛弯儿；而城市探索是深入城市的犄角旮旯的。Citywalk 是对一座城市的气质和灵魂的探访，而城市探索是在城市肉体中的深潜。北京的 citywalk 路线之多，主题之繁杂，以至于 STY 认为自己学校的十三个校区之间都能拉一个 citywalk 路线；而对城市探索的关注度就堪比人们对奥林匹克公园的林中的北纬四十度纪念碑一样。当然了，代价是你的腚。

# 设计师只用管规划路线就可以了，而骑行者要考虑的就多了

昌平绿道，a.k.a. 42 公里绿道，从东小口森林公园的正门开始。接着一路向北，穿过贺新公园，太平郊野公园等 STY 基本上没听说过的公园和绿地。之所以特地点出 “绿地”，是因为在某些区段上绿道确实就是在路边的小树林子里穿梭。在一段段被公园围栏保护的道路之外，还有大量需要自行寻找路线的部分，例如从奥北森林公园的南园骑到北园的方法 —— 你需要额外向东骑三公里到顺行方向上最近的红绿灯过马路。

到达定泗路后，你就进入了完全没有路线指示的地方了。除非打开缺德地图，你根本想不到你需要从旁边的镇子里传过去；除非停下来看施工信息公示栏，你根本想不到蓝色围挡的另一边就是温榆河，只不过现在正在修缮堤坝你下不去而已。

从临时桥过温榆河之后，你就要体验一大段原生态乡村公路了，堪比上次去清河营公园时候的那条。想走绿道的路线？想多了，刚才不是说了温榆河大堤在维修嘛，你过不去。无论如何，在成功绕出来之后你就能看到沿着沙河水库的公路，此时沿着路一直骑到看着昌平线高架桥的地方就行 —— 那里有一条通往昌平城区的小路。

这一部分之所以要取这个小标题，是因为…… 设计师们你们有没有骑过这条路线啊！如果要舒服的在这条路线上骑行，你至少需要：

  • 知道厕所的位置
  • 了解绿道上哪些区段有陡坡和急弯
  • 打探好两段绿道之间最近的红绿灯在哪里
  • 清楚沿途工地的数量

而很明显没有一个出现在路线图上。当然了，不能难为人家，但是您好歹把急转弯标的明显一点儿吧！因为向心力过小当场侧滑的事情咱也不是没经历过。

# 于是 STY 折磨了一辆共享单车

折磨一词用的相当精准。“折磨” 指的是作者骑车时比较狂野的风格与差强人意的路面材质，体现了作者对于城市探索的热爱和对骑车体验较差的遗憾，和昌平区基础建设蓬勃发展的背景相呼应，运用了夸张的修辞手法，使语句更有张力，更能吸引读者阅读兴趣。

# 清河

STY 从立水桥地铁站出发。看着赶着早八的叔叔阿姨，从那天起后面五十天不用赶早七 STY 的感到了由衷的舒畅。

扫完车车后第一步就是找到绿道的正式入口，东小口森林公园。路线也十分清晰 —— 至少看起来是 —— 就是沿着清河边上的巡河路骑到某座高尔夫球场后向北拐。What can possibly go wrong?

我真傻，真的，我单知道早八的时候人们要上班，会有很多车到城里面来；我不知道清河边也会有。我一清早起来就出了地铁站，拿手机扫了一辆共享单车，要去东小口森林公园。STY 是很听话的，导航的话句句听；他出去了。STY 就在清河滨河路骑车，准备在一座高尔夫球场前的路口拐弯。STY 往前面的路看，只见车排了一路，没有导航上显示的一路绿色了。他是不去其他路去骑的；各处去一看，果然到处都是车。STY 急了，央缺德地图出去寻新路。直到过一会儿，寻来寻去寻到旁边的清流路，看见路上竟是空的。STY 说，糟了，怕是大家想抄近道去京藏高速。再看地图；车车果然往上清桥开，前面的车都到延庆了，后面的车还在陈家营西路呢。

反正就是这么一回事，无论如何感谢迅哥和祥林嫂友情出演。STY 想不到在一个周一，在清河河堤上的清河路上，居然有这么多车。

沿途见得最多的除了清河自己，就是朝阳区和昌平区的界碑了。正当 STY 的朋友们晒自己和国界碑合影的时候，STY 对着北京俩区的界碑都兴奋的不行。不过有一说一，国界碑和朝阳区 - 昌平区界碑其实都是一个档次的，毕竟都是国务院刻的。

# 东小口森林公园


一靠近东小口森林公园，STY 就觉滴这次可能骑不开心了。本来想着这地方已经算得上是城乡结合部了，估计没啥人来，结果 —— 是哪个公司要来这儿团建啊！

当然了，对于他们给所有参与活动的人发自行车这件事儿，STY 根本没感觉，但是如果没看见那肯定更好，不过看见了问题也不大，当然了我朋友破防了，不是我，不过 —— 还是不知道这件事估计更好。


骑到第二个转弯，STY 就十分争气地侧滑摔了。旁边休息区里的老奶奶表示：

👩‍🦳：害，年轻人摔了没事儿，身体好的快～

（STY 爬起来并表示高度赞许）

STY 的小提示：自行车左边的把手控制前闸，右边的控制后闸，千万别搞混，而且刹车时要先刹前闸。如果先刹后闸，那你就会体验到上一段里 STY 的感受了。

东小口森林公园的布局很奇特。不知道的还以为 Georges-Eugène Haussmann 转世到 2008 年的北京，按他当年重建巴黎的图纸又来了一回。

42 公里绿道全程和各公园内的其他道路分离，这是难能可贵的一点。但问题在于 —— 绿道依然是人和自行车混行的啊！还好 STY 素质炒鸡高，每一次超车都响铃了。

不。这里是昌平，什么都可能发生，比如说桥前面是施工围挡，过去之后有一个初极狭，才通人的小门，进去后才是那个贺新公园。对了，门口的牌子上的 “自行车” 仨字儿明显是 2019 年绿道修到这里时候拿胶带挡上的。

# 太平郊野公园 与 奥北森林公园

从这里开始，路况就开始变得喜人了。一般情况下说的 “野” 林子分布在绿道两侧，给人一种待会儿包里就会有惊喜的感觉。是的，虫子会往下掉，别问 STY 是怎么知道的。

拜太平郊野公园独特的道路设计所赐，STY 成功错过了沿途的所有景点，包括那个传说中的 “中考体育练习场”：去这里纯粹是幸灾乐祸，毕竟 STY 那届体育中考全员满分（考试取消了）。点击这里回顾当时 STY 的记录。

出了门，就是太平庄中街。为了彰显 STY 高尚的驾车习惯，STY 向东骑了一点五公里到最近的红绿灯才过马路，而不是趁没车的时候直接 jaywalk。不对啊，好像全程车流没断过。
