# 第二站 清河营郊野公园

清河营郊野公园是顺着奥运会后修城市绿带的机会搞的小公园。这儿还是来广营的普法（注：和普鲁士与法兰西无关）公园，普法的主要形式是运动器材边上的法条说明和炸裂案例（记不清了所以建议您亲自康康吼）。

公园的结构分为南园和北园，和南边某个不愿透露姓名的森林公园儿一样，中间隔着一条原生态公路。南北两边还各有小湖，公园入口上的牌子悄悄告诉你说里面湖上有冰场。有高压电线塔在公园里面，看着挺赛博朋克的。请记住这段话……

# 第三站 清河慢行系统，或者什么其他名字


清河，某条堪称天通苑母亲河的小水沟海河支流北运河上段温榆河的支流。跨过这条河，您就进入■京市的城乡结合部了。

实名制羡慕，■运村地区只有某个不知名奥运场馆能比得上这里。




啊？■京市区界这么有排面，还是国务院立的碑！别人都是寒假出国顺手拍个中国界碑，咱拍个昌平区界，6！

有一说一，国务院立北京市区的碑也合情合理，毕竟北京不大也是个 municipality 嘛，等效一个省，北京的城区就相当于县一级啦。县以下的分界线就地方自己管了，STY 忘不了奥运村社区和来广营乡的分界线，那个纯纯是大号停车指示牌。

如何判断你来到了一个中国城市公园？

  • 路面崭新崭新的，尽管应该没人在竣工以后再管过它。
  • 路边的墙给人一种时间凝滞的体验（没人再刷过）。
  • 长椅：如果是铁质的，那么颜色是 #FFD700 ；如果铁架木质的，那木头是 #8B4513 的，铁架子是 #000000 的，而且刷漆的时候应该不和刷漆师傅之外的人说。
  • 自行车道叫什么 慢行系统 绿色廊道 之类特别像道法题里材料里出现的瞎总结式名词儿。
  • 塑胶路面 有的叫慢跑道，有的叫快跑道，有的叫自行车道，有的叫人行道。总之要么是红的要么是蓝的 —— 而且不掉色儿！
  • 水系：肯定有一条全国、全省、全市或者本区的非著名河流。
  • 鸭鸭：绝对有，而且数量较多，公鸭和母鸭占比失衡，小鸭鸭也很多。
  • 鱼鱼：和鸭鸭构成竞争关系。主要食物为馒头和公园河边买的鱼食儿。
  • 钓鱼佬：咱还用说嘛？
