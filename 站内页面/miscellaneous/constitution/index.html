# I 核心原则

  1. Cyberpark 为 STY 所有，不得转让。
  2. Cyberpark 上一切内容，除特殊说明外，属于 STY。
  3. STY 有自由操作 Cyberpark 的权利和义务

# II 更新

  1. 更新频率由 STY 决定。
  2. 原则上，在学期中每月更新一篇，在寒暑假每两周更新一篇；但在有特殊说明的情况下可以拖更，但不得超过三次预定的更新。
  3. 每个季度应对正文内容进行修缮，例如修改错字和语病、增加细节等。

# III 内容

  1. Cyberpark 的内容应与 STY 生活相关，或者虚构的故事。
  2. 当图片不是 STY 拍摄或制作的时候，应标出来源。
  3. 保证随时至少有一个备用图床。
  4. 上传图片时应压缩。
  5. 正文字数在 1000 字以上。
  6. 应及时更换颜色主题 。

# IV 标志与名称

  1. Cyberpark 的站标是黑 AC 和白 AC。
  2. Cyberpark 的站徽是红环 AC。
  3. 本站全称 Ameitsu Cyberpark，简称 The Cyberpark 或 Cyberpark，缩写为 AC。
  4. 本站的格言是 Ameitsu Gambatte!!!（注意斜体、加粗和感叹号数量）

# V 其他事项

  1. 站规可每月月底修改。
  2. 站规体系由站规和站规等效页面组成。对于站规等效页面的认定需明显标出。
